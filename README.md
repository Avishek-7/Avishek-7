## 📌 Featured Projects

### 🤖 AI Support Assistant
A **production-grade RAG-based customer support chatbot** built on a microservices architecture.
- **Features:** Multi-tenant RAG pipeline, semantic document search, conversation history, analytics dashboard, admin panel
- **Architecture:** 9 decoupled services — API Gateway, Auth, Document, Vector, RAG, LLM, Chat, Analytics, Admin
- **Stack:** FastAPI, Python, Qdrant, PostgreSQL, Redis, RabbitMQ, Docker Compose, Prometheus, Grafana, Jaeger
- **Repo:** [GitHub](https://github.com/Avishek-7/AI_customer_support)

---

### 🎙️ MeetingIntel
An **AI-powered meeting transcript analyzer** with local transcription and LLM-generated insights.
- **Features:** Audio transcription, action item extraction, meeting summaries, speaker analysis, dark-themed React dashboard
- **Stack:** FastAPI, Python, Anthropic SDK (Claude), faster-whisper (large-v3), React, TypeScript
- **Repo:** [GitHub](https://github.com/Avishek-7/meeting_intel.git)
